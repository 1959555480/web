# [html] HTML全局属性(global attribute)有哪些（包含H5）？
    全局属性: 用于任何HTML5元素的属性
    class
    contenteditable
    data-* 为元素增加自定义的属性
    id 元素id 文档内唯一
    lang 元素内容的语言
    style 行内css样式
    title 规定元素有关的额外信息
    translate 元素和子孙节点内容是否需要本地化
