# [css] 简述下你理解的优雅降级和渐进增强
    优雅降级
    先不考虑兼容，优先最新版本浏览器效果，之后再逐渐兼容低版本浏览器。

    渐进增强
    考虑兼容，以较低（多）浏览器效果为主，之后再逐渐增加对新版本浏览器的支持，以内容为主。也是多数公司所采用的方法。
