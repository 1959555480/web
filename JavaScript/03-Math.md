### 只保留整数部分（丢弃小数部分）

``` 
parseInt(5.1234);      // 5
```

### 向下取整

``` 
Math.floor(5.1234);     // 5    
```

### 向上取整

``` 
Math.ceil(5.1234);     // 6
```

### 四舍五入（小数部分）

``` 
Math.round(5.1234);      // 5
Math.round(5.6789);     // 6
```

### 取绝对值

``` 
Math.abs(-1);      // 1
```

### 返回两数中的较大者

``` 
Math.max(1,2);     // 2
```

### 返回两数中的较小者

``` 
Math.min(1,2);    // 1 
```

### 随机数（0-1）

``` 
Math.random();
```

### 开平方数

``` 
Math.sqrt(0.01)
```
